先到 GitHub 網站建立一個新的空倉庫，名稱可以也叫 BlazorTemplate。建立時不要勾 README、.gitignore、license，因為你本機已經有提交了。

git remote add origin https://github.com/ppk11654/BlazorTemplate.git
git branch -M main
git push -u origin main

先確認有沒有連成功，再補一行
git remote -v
